# Dataset

**Source:** [USA Real Estate Dataset — Kaggle](https://www.kaggle.com/datasets/ahmedshahriarsakib/usa-real-estate-dataset)

The CSV file (`realtor-data.csv`) is ~170MB and too large for GitHub. To reproduce the analysis:

1. Download the dataset from the Kaggle link above
2. Place `realtor-data.csv` in this directory
3. For SQL Server loading, copy it to a local path like `C:\SQLData\Dataset\realtor-data.csv`

**Records:** 2,226,382 raw → 1,481,944 after cleaning (33.4% removed)
